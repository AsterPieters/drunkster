# drunkster
DRUNKSTER, get drunk or DIE
A drinking game made by Aster Pieters.

To use drunkster:

    Linux:
        sudo apt install python3
        pip install pygame
        python3 drunkster.py

    Windows:
        install python 3.10 from playstore
        pip install pygame
        python drunkster.py

    Macos:
        Buy a pc
